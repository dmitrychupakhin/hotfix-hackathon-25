Есть запрос от заказчика на разработку программного обеспечения:
__ЗапросЗаказчика__
Тебе необходимо составить план работы над программным обеспечением, которое требует заказчик. План должен состоять из 3-7 этапов, которые характеризуются названием,
датой начала (__ДатаНачалаПлана__) и датой конца (от 15 до 70 дней в зависимости от требований заказчика). Даты этапов могут пересекаться, могут не пересекаться.
Ты должен вернуть строго только json в таком формате:
{
    "project_plan": [
        {
            "name": "Название этапа 1",
            "start_date": "2025-03-06",
            "end_date": "2025-03-10"
        },
        {
            "name": "Название этапа 2",
            "start_date": "2025-03-07",
            "end_date": "2025-03-17"
        }
    ]
}
