Тебе необходимо составить график проектной работы на основе следующих параметров:
Срок проекта: __СрокПроекта__
Запрос заказчика: __ЗапросЗаказчика__
Начало работы проекта: __ДатаНачалаПроекта__
Стек комадны: __СтекКоманды__
Придумай от 3 до 7 этапов для разработки и напиши для них даты (даты разных этапов могут накладыватьсяя). 
Дата начала плана должна быть, которую я указал выше, дата окончания последнего плана должна быть равна   __ДатаКонцаПроекта__
Ты должен вернуть строго только json в таком формате:
{
    "project_plan": [
        {
            "name": "Название этапа 1",
            "start_date": "2025-03-06",
            "end_date": "2025-03-10"
        },
        {
            "name": "Название этапа 2",
            "start_date": "2025-03-07",
            "end_date": "2025-03-17"
        }
    ]
}
