Есть несколько команд, которые характеризуются номером и их стэком:
{
    "team": 1, 
    "stack": "HTML, js, typescript, fast-api, docker",
    "last_end_date": "2025-03-06"
}
Есть запрос от заказчик:
__ЗапросЗаказчика__
Тебе необходимо выбрать ровно одну команду, у которой  наиболее подходящий стэк программ. 
Если таких несколько выбери команду с наименьшим полем last_end_date.
После того, как ты выберишь команду, тебе необходимо составить план работы над программным обеспечением, которое требует заказчик. План должен состоять из 3-7 этапов, которые характеризуются названием,
датой начала (это поле last_end_date выбранной команды) и датой конца (от 15 до 70 дней). Даты этапов могут пересекаться, могут не пересекаться.
Ты должен вернуть строго только json в таком формате (predict_team - номер выбранной тобой команды):
{
    "predict_team": 3,
    "project_plan": [
        {
            "name": "Название этапа 1",
            "start_date": "2025-03-06",
            "end_date": "2025-03-10"
        },
        {
            "name": "Название этапа 2",
            "start_date": "2025-03-07",
            "end_date": "2025-03-17"
        }
    ]
}
